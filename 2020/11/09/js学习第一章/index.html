js学习第一章

完整的JavaScript包括以下几部分：

  • 核心（ECMAscript）
  • 文档对象模型（DOM）
  • 浏览器对象模型（BOM）

ECMAscript

即是ECMA-262定义的语言，并不限于web浏览器上。web浏览器是为其提供实现和与环境交互的扩展（如DOM）。ECMAscript只是对实现这个规范描述的所有方面的一门语言的称呼，而js实现了ECMAscript。

ECMA-262阐述了什么是ECMAscript符合性，要使其实现要满足

  • 支持ECMA-262中所描述的所有“类型、值、对象、属性和函数，以及程序语法及语义”
  • 支持Unicode字符标准

DOM

文档对象模型（Document Object Model）是一个应用程序接口（API）。

DOM并非只能通过js访问。DOM就是用ECMAscript实现的，也成为了js的一大组成部分。

BOM

浏览器对象模型

js是一门用来与网页交互的脚本语言

  • 核心（ECMAscript）：由ECMA-262定义并提供核心功能
  • 文档对象模型（DOM）：提供与网页内容交互的方法和接口
  • 浏览器对象模型（BOM）：提供与浏览器交互的方法和接口
